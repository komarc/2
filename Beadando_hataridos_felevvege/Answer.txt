## Copilot
Természetesen, örömmel segítek! Az alábbiakban lépésről lépésre bemutatom, hogyan hozhatod létre az adatbázist az sqliteonline.com oldalon:
- Táblák létrehozása: Először is létre kell hoznod az összes szükséges táblát. Az alábbi SQL parancsokat használhatod erre:

CREATE TABLE Páciensek (
    PatientID INTEGER PRIMARY KEY,
    Név TEXT NOT NULL,
    Születési_dátum DATE NOT NULL,
    Cím TEXT NOT NULL
);

CREATE TABLE Orvosok (
    DoctorID INTEGER PRIMARY KEY,
    Név TEXT NOT NULL,
    Specializáció TEXT NOT NULL
);

CREATE TABLE Látogatások (
    VisitID INTEGER PRIMARY KEY,
    PatientID INTEGER,
    DoctorID INTEGER,
    Dátum DATE NOT NULL,
    Megjegyzések TEXT,
    FOREIGN KEY (PatientID) REFERENCES Páciensek (PatientID),
    FOREIGN KEY (DoctorID) REFERENCES Orvosok (DoctorID)
);

CREATE TABLE Receptek (
    PrescriptionID INTEGER PRIMARY KEY,
    VisitID INTEGER,
    Gyógyszer TEXT NOT NULL,
    Adagolás TEXT NOT NULL,
    FOREIGN KEY (VisitID) REFERENCES Látogatások (VisitID)
);

CREATE TABLE Fizetések (
    PaymentID INTEGER PRIMARY KEY,
    VisitID INTEGER,
    Dátum DATE NOT NULL,
    Összeg INTEGER NOT NULL,
    FOREIGN KEY (VisitID) REFERENCES Látogatások (VisitID)
);

- Adatok feltöltése: Miután létrehoztad a táblákat, feltöltheted őket tesztadatokkal. Az alábbi SQL parancsokat használhatod erre:

INSERT INTO Páciensek (PatientID, Név, Születési_dátum, Cím) VALUES (1, 'Kovács Béla', '1970-05-01', 'Budapest');
INSERT INTO Orvosok (DoctorID, Név, Specializáció) VALUES (1, 'Dr. Szabó László', 'Háziorvos');
INSERT INTO Látogatások (VisitID, PatientID, DoctorID, Dátum, Megjegyzések) VALUES (1, 1, 1, '2024-05-08', 'Kontroll');
INSERT INTO Receptek (PrescriptionID, VisitID, Gyógyszer, Adagolás) VALUES (1, 1, 'Aspirin', 'Napi 1x');
INSERT INTO Fizetések (PaymentID, VisitID, Dátum, Összeg) VALUES (1, 1, '2024-05-08', 5000);

- Lekérdezések: Végül, miután feltöltötted az adatbázist adatokkal, elvégezheted a különböző lekérdezéseket. Az alábbi SQL parancsokat használhatod erre:

-- Melyik páciens melyik orvosnál járt?
SELECT P.Név, O.Név
FROM Páciensek P
JOIN Látogatások L ON P.PatientID = L.PatientID
JOIN Orvosok O ON L.DoctorID = O.DoctorID;

-- Melyik páciensnek milyen gyógyszert írtak fel?
SELECT P.Név, R.Gyógyszer
FROM Páciensek P
JOIN Látogatások L ON P.PatientID = L.PatientID
JOIN Receptek R ON L.VisitID = R.VisitID;
